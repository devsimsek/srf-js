Roadmap And Future Plans

SRF project aims to be more than just a simple router. This will be a framework which you can generate ui without actually designing stuff. I plan to implement this feature using bootstrap, tailwind and my own little css framework.

SRF will be all modular. You will choose what features would you like to include on your app using @srf/cli (soon...)

SRF is not only a web framework. My goal is to make SRF multiplatform and able to run applications using your mobile devices and more!
